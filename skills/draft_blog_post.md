# Draft Blog Post Workflow

## When to use
Run this workflow when asked to draft a technical blog post or announcement.

## Workflow
1. Determine the target audience (technical vs business).
2. Outline the core narrative and key takeaways.
3. Write an engaging introduction.
4. Develop the body paragraphs with technical accuracy.
5. Add a conclusion with a strong call to action.

## Output
A well-structured markdown document containing the blog post draft.
